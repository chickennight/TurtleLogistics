# Topic convention
```
(수정) / 역할 / 기기명 / 값 
수정 : mod
역할 : ord,div,web
기기명 : sch, veri, cam, servo
값 : power, interval, speed, distance,info,res
```
```
A가 B에게 보낸다, A가 나(B)에게 보낸다
=>/A역할/B역할/B기기명/value
=>/sup/div/veri/info
```
